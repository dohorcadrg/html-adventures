What is this?

This is an adventure game that asks HTML, CSS and Javascript questions. It aims to test your knowledge in those areas while entertaining you with a story.

Why is this so complicated?

I like a challenge. Also, an HTML adventure game is something that is exciting, and educational at the same time. The idea itself is complicated, but the ending results are worth it.

Where is the CSS and Javascript Section?

Due to the recent results of the 2016 election, the CSS and Javascript sections are unavailable. Also due to time constraints, the CSS and JS sections are unable to be completed.

Why is there a CSS question in the HTML section?

Why not? It is mainly because HTML and CSS are much related to each other. HTML determines the structure of the webpage, while CSS determines how it is displayed. They are so related, it is sometimes hard to differentiate between a HTML and a CSS question.

Where are the other questions?

Gone. In all seriousness; Due to time constraints, certain sections have to be shorten off, cut off, or deleted completely.
